# End-to-End ETL & Data Warehouse System

A cloud-based data engineering project that ingests e-commerce data, transforms it through ETL pipelines, and loads it into a structured data warehouse for analytics and reporting.

## Tech Stack

- Python
- SQL
- AWS S3
- Amazon Redshift
- Apache Airflow
- Power BI

## Dataset

Brazilian E-Commerce Public Dataset by Olist
